HTML e CSS

A Internet utiliza uma linguagem que permite formatar as páginas web que visitamos. HTML significa Hiper Text Markup Language e é uma linguagem utilizada para criar páginas com hipertexto (com hiperligações). Os sinais de menor e de maior <> delimitam as tags (etiquetas). O símbolo / permite fechar a tag.
